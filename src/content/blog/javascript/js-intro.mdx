Loops let you repeat a block of code without writing the same logic again and again.

## Why loops matter

You use loops when you want to:

- go through a list of values
- repeat work until a condition becomes false
- process data in a predictable pattern

## The `for` loop

Use a `for` loop when you know how many times something should run.

```javascript
for (let i = 0; i < 5; i++) {
  console.log(i);
}
```

This prints the numbers `0` through `4`.

## The `while` loop

Use a `while` loop when the number of iterations depends on a condition.

```javascript
let count = 0;

while (count < 3) {
  console.log(count);
  count++;
}
```

The loop keeps running as long as `count < 3` is true.

## The `for...of` loop

JavaScript also gives you `for...of` for iterating over arrays and other iterable values.

```javascript
const topics = ["loops", "arrays", "functions"];

for (const topic of topics) {
  console.log(topic);
}
```

This is often cleaner than manually using an index when you only care about the value.

## Final thought

If you know the count, start with `for`. If the condition controls the loop, use `while`. If you are iterating over an array, `for...of` is usually the easiest to read.
