Loops allow repeating operations.

## Example

```python
for i in range(5):
    print(i)
    ```

There are different type of loop like for loop, while loop.
for loops are used where the number of iteration is known beforehand, while loops are used where the number of iteration is not known beforehand.

## For Loop

```python
for i in range(5):
    print(i)
```

In this example, the for loop iterates over a sequence of numbers generated by the `range()` function, which produces numbers from 0 to 4. The loop executes the block of code (printing the current value of `i`) for each number in the sequence.
