Loops help you repeat work in a clean and readable way.

## The `for` loop

In Python, a `for` loop is commonly used when you want to iterate over a sequence.

```python
for i in range(5):
    print(i)
```

This prints the numbers `0` to `4`.

## The `while` loop

Use a `while` loop when the loop should continue until a condition changes.

```python
count = 0

while count < 5:
    print(count)
    count += 1
```

This loop keeps running until `count` reaches `5`.

## When to use which one

- Use `for` when iterating over a range, list, tuple, or string.
- Use `while` when the number of iterations is not fixed ahead of time.

## Example with a list

```python
languages = ["Python", "JavaScript", "C++"]

for language in languages:
    print(language)
```

This is one of the most common looping patterns in Python.
