Object-Oriented Programming, or OOP, is a way of organizing code using classes and objects.

## What is a class

A class is a blueprint.

It defines what data an object should store and what actions it can perform.

```python
class Student:
    def __init__(self, name):
        self.name = name
```

## What is an object

An object is an instance of a class.

```python
student = Student("Ishant")
print(student.name)
```

Here, `student` is an object created from the `Student` class.

## Why OOP is useful

- It groups related data and behavior together.
- It makes large programs easier to organize.
- It helps you reuse code through classes and methods.

## Final thought

Start by understanding classes, objects, and constructors. Once those feel natural, concepts like inheritance and polymorphism become much easier to learn.
