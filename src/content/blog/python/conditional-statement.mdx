Conditional statements help your program make decisions.

## The `if` statement

Use `if` when you want code to run only when a condition is true.

```python
age = 18

if age >= 18:
    print("You can vote.")
```

## The `elif` statement

Use `elif` when you want to check another condition after the first one fails.

```python
marks = 75

if marks >= 90:
    print("Grade A")
elif marks >= 60:
    print("Grade B")
else:
    print("Keep practicing")
```

## The `else` statement

`else` runs when none of the earlier conditions are true.

It acts like the fallback branch in your decision tree.

## Final thought

Whenever your program needs to choose between different outcomes, `if`, `elif`, and `else` are the tools you reach for first.
